객체(object)란?

            
1. 이름(key)와 값(value)로 구성된 속성을 담아놓은 컨테이너이다.

            
2. 객체에 변수를 선언하면 속성(property)을 사용할 수 있다. ex)document.getElementById('아이디명')

            
3. 객체에 함수를 선언하면 메서드(method)를 사용할 수 있다. ex)document.wirte('내용');
