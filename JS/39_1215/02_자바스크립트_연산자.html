02_자바스크립트_연산자

        자바스크립트 연산자는 사용자가 조건식을 만들어서 원하는 결과를 출력하고자 할 때 사용하는 기호이다. 
        
        
1.산술연산자
 : +, -, *, /, %(나머지값 )
        
        

        
2.문자열 결합연산자
 : +(문자+문자, 문자+숫자 결합시 사용)
        
        


        
비교연산자
 : 
        >, <, =>, <=, ==(true check), ===
        ex) 홈페이지 스크롤 내릴때 사진이 바뀌는것 
        화면스크롤값 scroll 변수선언 : let scroll; scroll>=300
        
        

        대입연산자 : =, +=, -=
        = : ex) a=b a를 b에 대입한다
        
        

        증감연산자 : ++(1씩 증가), --(1씩 감소)
        
        

        논리연산자 : &&(and), ||(or), !(not)
        
        


        삼항조건연산자 : 조건식 조건이 맞을때 출력값 1(true): 조건이 맞지 않을때 출력값 2(false)
        
조건식?실행문1:실행문2;

        
        

        타입연산자 : typeof(데이터 타입을 구분하고 형식을 알려줌)
        
        


        실습예제
        1. 사용자가 입력한 아이디가 'master', 'abcd', 'naver'중 하나만 입력해도 인증이 되며, 패스워드는 '1,2,3,4'일때 인증이 되도록 결과값이 나오도록 함 
        2. 결과값 '인증완료', '인증실패 다시시도하세요'
        조건식?인증완료:인증실패
