01_변수의 개념과 활용
변수(variable)란? 변하는 데이터를 저장하기 위한 일종의 메모리공간 이다.
        변수에 대입된 값은 항상 마지막에 저장된 값이 출력이 된다.

        1. 변수의 종류
        
            1) 전역변수 : 문서전체에 영향을 주는 변수
            * 팀작업시 문서 전체에 영향을 줄수있어서 조심하게 사용
        ex) 문화상품권 -> 전국 사용가능 
        let a = 10; (전역변수)

        2) 지역변수 : 함수(function)내에서 선언되어 사용되는 변수로서 그 함수 지역에서만 영향력을 가지는 변수(지역화폐개념)
            ex) 서울시지역화폐 -> 서울만사용가능 
            function 함수명(){let b = 5; //지역변수}
        
2. 변수(변수명:식별자)의 선언방법
변수 선언시 'var', 'let',
        'const', 키워드를 변수명 앞에 붙여주어 선언한다.  
        
        자바스크립트 최신버전 es6에서는 'let', 'const'사용을 추천한다
        var 변수명; 또는 let 변수명; 또는 const 변수명;

        var 변수명=값; var a=10; let b=5; const c=1000;
        
3. 변수명 작성시 주의사항
변수명을 작성하고자 할 때는 아래사항을 주의해야 한다. 
        

            1. 변수명은 영문 대, 소문자로 작성되어야 한다. 
                ex) var a; var A;

            2. 변수의 첫글자는 숫자, 기호가 올 수 없다 
                ex) 01A, 01_Aa

            3. 변수명 사이에는 스페이스(공백)을 사용할 수 없다. 
            ex) count 01

            4. 변수명은 색상이 변하는 단어(예약어)는 사용할 수 없다 
            ex) var, if, break등 이미지 참고

            5. 변수명에는 특수기호를 사용할 수 없다. (단, $와 '_'는 사용가능) ex) $Aa, 01_Aa

            6. 변수명은 의미를 담을 수 있는 이름으로 지정하고, 일명 낙타몽(카멜표기법)표기법으로 작성할 것을 권장한다. 
            ex) imagename(x) imageName(o)
