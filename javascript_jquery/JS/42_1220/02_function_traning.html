02_함수실습예제 - 자신의 신장, 몸무게를 통해 BMI 지수 구하기

자신의 키(신장), 몸무게(체중)을 입력하여 적정 체중 구하기

  1. 신장(m)을 입력하기 위한 명령프롬프트와 체중(kg)을 입력하기 위한 명령프롬프트 사용
  2. 함수를 작성하여 신장, 체중값을 사용하여 계산식 작성하기
  3. 연산된 결과 값을 통해 적정체중인지를 비교(if문)하여 결과를 출력한다.

BMI = 체중(KG) / (신장(m) * 신장(m))

저체중은 배경색 주황색, 정상은 녹색, 과체중은 파랑, 비만은 보라색, 고도비만은 빨강색 배경으로 서식을 지정함
