            typeof - typeof연산자는 사용자가 입력한 데이터가 어떤 종류의 형식인지를 알려주는 연산자이며, 변수앞에 numberm string등을 작성하면 실제 데이터 형 식을 알 수 있다. 

            es6문법에서 새롭게 추가된 기호 => 템플릿 리터럴(`): 연산자를 사용하지 않아도 간단한 방법으로 새로운 문자열을 삽입할 수 있는 기능을 제공한다. 이를 문자열 인터폴레이션(string interpolation)이라 한다.
